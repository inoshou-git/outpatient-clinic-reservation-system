一般ユーザー向け操作マニュアル

1. システム概要

このシステムは、外来診療、訪問診療、通所リハ会議など、様々な予約を一元的に管理するための予約システムです。ユーザーは自身の役割に応じた機能を利用し、効率的な予約管理を行うことができます。

2. 概要

このマニュアルは、外来診療予約システムの一般ユーザー向け機能について説明します。一般ユーザーは、予約の追加・編集・削除、予約状況の確認が可能です。

3. ログイン

システムにアクセスするには、ユーザーIDとパスワードを使用してログインしてください。

  1. システムURLにアクセスします。
  2. ログイン画面で、ユーザーIDとパスワードを入力し、「ログイン」ボタンをクリックします。
  3. 初回ログイン時には、新しいパスワードの設定が求められます。画面の指示に従ってパスワードを変更してください。

4. 予約管理

予約の追加、更新、削除が可能です。操作中はローディングアイコンが表示されます。

4.1. 新規予約登録

  1. ログイン後、ヘッダーメニューから「新規予約登録」を選択するか、カレンダー/週表示の空き時間枠をクリックします。
  2. 表示されるフォームで、まず「予約種別」を選択します。
    • 外来診療: 患者ID（数字のみ）、患者名、日付、時間、診察内容を入力します。
    • 訪問診療: 施設名（任意）、日付、開始時間、終了時間（必須）、診察内容を入力します。
    • 通所リハ会議: 日付、開始時間、終了時間（必須）を入力します。
  3. 「関係者にメールで通知する」チェックボックスをオンにすると、システムに登録されている全ユーザーに通知メールが送信されます。
  4. 「予約を登録する」ボタンをクリックして予約を保存します。

4.2. 予約の編集

  1. 予約一覧またはカレンダー/週表示で、編集したい予約をクリックします。
  2. 表示されるフォームで、予約種別に応じた項目を編集します。
    • 外来診療: 患者ID、患者名、日付、時間、診察内容。
    • 訪問診療: 施設名、日付、開始時間、終了時間、診察内容。
    • 通所リハ会議: 日付、開始時間、終了時間。
  3. 「関係者にメールで通知する」チェックボックスをオンにすると、変更内容が記載された通知メールが送信されます。
  4. 「予約を更新する」ボタンをクリックして変更を保存します。

4.3. 予約の削除

  1. 予約一覧で、削除したい予約の行にある「削除」アイコンをクリックします。
  2. 確認ダイアログが表示されるので、「OK」をクリックすると予約が削除されます。
  3. 予約削除時にも通知メールが送信されます。

5. 予約状況の確認

5.0. 予約リストの見方

予約リストでは、以下の項目が表示されます。

  • 日付: 予約の日付。
  • 時間: 外来診療の場合は予約時間、訪問診療・通所リハ会議の場合は開始時間 - 終了時間。
  • 予約種別: 外来診療、訪問診療、通所リハ会議のいずれか。
  • 患者ID/施設名: 外来診療の場合は患者ID、訪問診療の場合は施設名。通所リハ会議の場合は表示されません。
  • 患者名: 外来診療の場合の患者名。訪問診療・通所リハ会議の場合は表示されません。
  • 診察内容: 外来診療・訪問診療の場合の診察内容。通所リハ会議の場合は表示されません。
  • 最終更新者: 予約を最後に更新したユーザー。
  • 操作: 予約の編集、削除を行うためのボタン。

予約一覧、カレンダー表示、週表示で予約状況を確認できます。

  • **予約リスト:** 登録されている予約を一覧で確認できます。
  • **月表示:** 月単位で予約状況をカレンダー形式で確認できます。
  • **週表示:** 週単位で予約状況をカレンダー形式で確認できます。週表示では、「午前診療」「午後診療」「全て表示」ボタンで表示する時間帯を切り替えることができます。

6. その他の機能

  • **ローディング表示:** 各操作の実行中は、画面全体にローディングアイコンが表示され、処理中であることが視覚的にわかります。
