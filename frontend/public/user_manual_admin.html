管理者向け操作マニュアル

1. 概要

このマニュアルは、外来診療予約システムの管理者向け機能について説明します。管理者アカウントでは、ユーザー管理、予約の追加・編集・削除、予約不可時間帯の設定など、システム全体の運用に関わる重要な操作が可能です。

2. ログイン

システムにアクセスするには、管理者アカウントのユーザーIDとパスワードを使用してログインしてください。

  1. システムURLにアクセスします。
  2. ログイン画面で、管理者アカウントのユーザーIDとパスワードを入力し、「ログイン」ボタンをクリックします。
  3. 初回ログイン時には、新しいパスワードの設定が求められます。画面の指示に従ってパスワードを変更してください。

3. ユーザー管理

管理者のみがユーザーの作成、編集、削除を行うことができます。

3.1. 新規ユーザー作成

  1. ログイン後、ヘッダーメニューから「ユーザー管理」を選択します。
  2. 「新規ユーザー作成」ボタンをクリックします。
  3. 表示されるフォームに、ユーザーID、氏名、所属部署、メールアドレス、権限（一般、閲覧、管理者）を入力します。
  4. 「作成」ボタンをクリックすると、ユーザーが作成され、自動生成された初期パスワードがユーザーのメールアドレスに送信されます。
  5. 操作中はローディングアイコンが表示されます。
注意: 新規作成されたユーザーは、初回ログイン時にパスワードの変更が強制されます。

3.2. ユーザー情報の編集

  1. 「ユーザー管理」画面で、編集したいユーザーの行にある「編集」アイコンをクリックします。
  2. 表示されるフォームで、氏名、所属部署、メールアドレス、権限を編集します。ユーザーIDは変更できません。
  3. 「更新」ボタンをクリックして変更を保存します。
  4. 操作中はローディングアイコンが表示されます。

3.3. ユーザーの削除

  1. 「ユーザー管理」画面で、削除したいユーザーの行にある「削除」アイコンをクリックします。
  2. 確認ダイアログが表示されるので、「OK」をクリックするとユーザーが削除されます。
  3. 操作中はローディングアイコンが表示されます。

4. 予約管理

予約の追加、更新、削除が可能です。操作中はローディングアイコンが表示されます。

4.1. 新規予約登録

  1. ヘッダーメニューから「新規予約登録」を選択するか、カレンダー/週表示の空き時間枠をクリックします。
  2. 表示されるフォームに、患者ID（数字のみ）、患者名、日付、時間、診察内容を入力します。
  3. 「関係者にメールで通知する」チェックボックスをオンにすると、システムに登録されている全ユーザーに通知メールが送信されます。
  4. 「予約を登録する」ボタンをクリックして予約を保存します。

4.2. 予約の編集

  1. 予約一覧またはカレンダー/週表示で、編集したい予約をクリックします。
  2. 表示されるフォームで、患者ID、患者名、日付、時間、診察内容を編集します。
  3. 「関係者にメールで通知する」チェックボックスをオンにすると、変更内容が記載された通知メールが送信されます。
  4. 「予約を更新する」ボタンをクリックして変更を保存します。

4.3. 予約の削除

  1. 予約一覧で、削除したい予約の行にある「削除」アイコンをクリックします。
  2. 確認ダイアログが表示されるので、「OK」をクリックすると予約が削除されます。
  3. 予約削除時にも通知メールが送信されます。

4.4. 予約の一括操作

予約一覧画面で「一括操作」ボタンをクリックすると、複数の予約を選択して一括で削除できます。

  1. 「一括操作」ボタンをクリックします。
  2. 各予約の行に表示されるチェックボックスをオンにして、削除したい予約を選択します。
  3. 「選択した項目を削除」ボタンをクリックします。
  4. 確認ダイアログが表示されるので、「OK」をクリックすると選択した予約が削除されます。

5. 予約不可設定管理

特定の時間帯や終日を予約不可として設定できます。操作中はローディングアイコンが表示されます。

5.1. 新規予約不可設定

  1. ヘッダーメニューから「予約不可設定」を選択します。
  2. 「新規予約不可設定」ボタンをクリックします。
  3. 表示されるフォームで、開始日、終了日（終日の場合）、開始時間、終了時間（時間指定の場合）、理由を入力します。
  4. 「関係者にメールで通知する」チェックボックスをオンにすると、システムに登録されている全ユーザーに通知メールが送信されます。
  5. 「登録」ボタンをクリックして設定を保存します。

5.2. 予約不可設定の編集

  1. 予約不可設定一覧で、編集したい設定の行にある「編集」アイコンをクリックします。
  2. 表示されるフォームで、設定内容を編集します。
  3. 「関係者にメールで通知する」チェックボックスをオンにすると、通知メールが送信されます。
  4. 「更新」ボタンをクリックして変更を保存します。

5.3. 予約不可設定の削除

  1. 予約不可設定一覧で、削除したい設定の行にある「削除」アイコンをクリックします。
  2. 確認ダイアログが表示されるので、「OK」をクリックすると設定が削除されます。
  3. 予約不可設定削除時にも通知メールが送信されます。

5.4. 祝日の一括登録

日本の祝日データを一括で予約不可設定として登録できます。

  1. 「予約不可設定管理」画面で「祝日を一括登録」ボタンをクリックします。
  2. 確認ダイアログが表示されるので、「OK」をクリックすると祝日が登録されます。
  3. 操作中はローディングアイコンが表示されます。

6. その他の機能

  • **カレンダー/週表示:** 予約状況をカレンダー形式や週単位で確認できます。
  • **ローディング表示:** 各操作の実行中は、画面全体にローディングアイコンが表示され、処理中であることが視覚的にわかります。
