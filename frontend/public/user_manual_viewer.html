閲覧者向け操作マニュアル

1. 概要

このマニュアルは、外来診療予約システムの閲覧者向け機能について説明します。閲覧者は、予約状況の確認のみが可能です。

2. ログイン

システムにアクセスするには、提供されたユーザーIDとパスワードを使用してログインしてください。初回ログイン時には、セキュリティのため新しいパスワードの設定が求められます。

3. 予約画面の見方と新機能

予約状況は「週」「月」「リスト」の3つの形式で確認できます。画面上部のボタンで切り替えます。

✨ 主な新機能と変更点 (2025年8月版)

リアルタイム更新
他のユーザーが予約を変更すると、あなたの画面も自動的に更新されます。ページを再読み込みする必要はありません。
予約種別ごとの色分け (週表示)
週表示カレンダーで、予約の種類によって背景色が変わります。
■ 水色: 外来診療 / ■ 緑色: 訪問診療 / ■ 紫色: 通所リハ会議
予約不可ルールの追加
毎週水曜日の午後 (13:00以降) は、全ての予約ができなくなりました。カレンダー上ではグレーアウトして表示されます。
カレンダー表示の改善 (月表示)
月表示カレンダーで、土日の日付がグレーアウトされるようになり、週末が視覚的に分かりやすくなりました。

4. 予約の確認

予約一覧、カレンダー、週表示で現在の予約状況を確認できます。

  • カレンダー/週表示: 予約状況をカレンダー形式や週単位で確認できます。週表示では、「午前診療」「午後診療」「全て表示」ボタンで表示する時間帯を切り替えることができます。
  • 予約リスト: 予約の詳細情報を一覧で確認できます。
